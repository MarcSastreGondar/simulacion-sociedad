Este proyecto se ha ejecutando usando Visual Studio Code, por ende, no se garantiza el funcionamiento correcto de la aplicación al ejecutarlo desde cualquier otra aplicación.

Prerrequisitos:
1.Tener Python ya instalado con la versión 3.12.10 (no se garantiza el funcionamiento correcto en otras versiones)


Todos los comandos necesarios para crear y configurar el entorno virtual. Ejecutarlos en orden:
1. Crear el entorno virtual
python -m venv venv

2. Activar el entorno virtual
#    Windows:
venv\Scripts\activate

#    Linux / macOS:
source venv/bin/activate

3. Actualizar pip a la versión más reciente
python -m pip install --upgrade pip

4. Instalar todas las librerías necesarias
pip install "mesa[rec]" numpy pandas matplotlib seaborn solara tqdm ipywidgets
pip install "starlette<0.46" --force-reinstall

5. Verificar que todas las librerías se instalaron correctamente
pip list



Cómo usar este entorno después de crearlo y cómo ejecutar:
1. Activar el venv
venv\Scripts\activate     # Windows
source venv/bin/activate  # Linux / macOS:

2. En el notebooks/simulacionSociedad.ipynb:
    2.1 Abrir el archivo
    2.2 Pulsar el botón de "Select Kernel" y elegir el venv que hemos creado
    2.3 Pulsar el botón de "Restart Kernel" y darle al botón de "Run All"
    2.3 En el apartado de visualización, usar la interfaz gráfica del Cell Output para realizar la ejecución que se quiera.


Puede ser necesario seleccionar la versión 3.12.10 como Intérprete de Python en VS Code:
    1. Ctrl + Shift + P
    2. Seleccionar "Python: Select Interpreter"
    3. Seleccionar el venv que hemos creado
