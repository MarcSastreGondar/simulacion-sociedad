Este proyecto se ha ejecutando usando Visual Studio Code, por ende, no se garantiza el funcionamiento correcto de la aplicación al ejecutarlo desde cualquier otra aplicación.

Prerrequisitos:
-Tener Python ya instalado


Todos los comandos necesarios para crear y configurar el entorno virtual. Ejecutarlos en orden:
1. Crear el entorno virtual
python -m venv venv

2. Activar el entorno virtual
#    Windows:
venv\Scripts\activate

#    Linux / macOS:
source venv/bin/activate

3. Actualizar pip a la versión más reciente
python -m pip install --upgrade pip

4. Instalar todas las librerías necesarias
pip install "mesa[rec]" numpy pandas matplotlib seaborn solara tqdm ipywidgets jupyterlab
pip install "starlette<0.46" --force-reinstall

5. Verificar que todas las librerías se instalaron correctamente
pip list



Cómo usar este entorno después de crearlo y cómo ejecutar:
1. Activar el venv
venv\Scripts\activate     # Windows
source venv/bin/activate  # Linux / macOS:

2. Ejecutar Jupyter Lab
jupyter lab

3. En el Jupyter Lab:
    3.1 Abrir el archivo notebooks/simulacionSociedad.ipynb
    3.2 Pulsar el botón de "Restart the kernel and run all cells" (en caso de querer volver a ejecutar el archivo entero)
    3.3 En el apartado de visualización, usar la interfaz gráfica del Cell Output para realizar la ejecución que se quiera.
