Este proyecto se ha ejecutando usando Visual Studio Code, por ende, no se garantiza el funcionamiento correcto de la aplicación al ejecutarlo desde cualquier otra aplicación.

Prerrequisitos:
-Tener Python ya instalado


Todos los comandos necesarios para crear y configurar el entorno virtual. Ejecutarlos en orden:
1. Crear el entorno virtual
python -m venv venv

2. Activar el entorno virtual
#    Windows:
venv\Scripts\activate

#    macOS / Linux:
source venv/bin/activate

3. Actualizar pip a la versión más reciente
python -m pip install --upgrade pip

4. Instalar todas las librerías necesarias
pip install mesa numpy pandas matplotlib seaborn solara tqdm

5. Instalar versiones específicas estables
pip install mesa>=3.5.0 numpy pandas matplotlib seaborn solara tqdm networkx

6. Verificar que todas las librerías se instalaron correctamente
pip list



Cómo usar este entorno después de crearlo:
1. Elegirlo como intérprete de Python en el Visual Studio Code. 
    1.1 Pulsar Ctrl + Shift + P,.
    1.2 Escribir "Python: Select Interpreter".
    1.3 Seleccionar la opción que tenga el venv que hemos instalado.

2. Dentro del archivo simulacionSociedad.ipynb, pulsar el botón que pone "Select Kernel" localizado arriba a la derecha, elegir la opción "Python Environments..." y seleccionar el venv que 
acabamos de crear, que probablemente tendrá asociado un icono de una estrella.
